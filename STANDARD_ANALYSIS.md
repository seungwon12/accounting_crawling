## 4. 수정 이력

### 2026-03-16 (이전 대화)
1. **TOC level 소수점 처리**: `int()` → `float()` 변환
2. **재시도 로직 스코프 버그**: `all_set` 필터로 `all_standards` 범위 내 실패만 재시도

### 2026-03-18 (이번 대화)
1. **CF 부모 섹션 누락 수정**: `toc_parser.py` — `include_parents` 파라미터 추가 → CF 725개 문단 수집 (이전 615개)
2. **div.para + div.idt-1 구조 처리**: `section_parser.py` — 케이스 2 분기 추가 (B 유형)
3. **config.py 수정**: 1117 추가, 4xxx 제거, 누락 2xxx 10개 추가, SPECIAL_DOCUMENTS 추가
4. **orchestrator.py 수정**: 특수문서 URL/유형 매핑, 크롤링 대상 목록에 특수문서 포함
